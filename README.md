# Next.js バズアプリ・テンプレート

このテンプレートは、SNSでバズるための「ミニゲーム」「診断アプリ」「ジェネレーター」を爆速で構築するためのNext.jsボイラープレートです。

## 特徴
- **Next.js 15+ (App Router)**: 最新の構成。
- **Tailwind CSS 4.0**: 高速なスタイリング環境。
- **Vercel OG (API Route)**: SNS共有時に必須の「動的OG画像」を自動生成。
- **SNS共有機能**: X（旧Twitter）への共有ボタンを標準搭載。
- **パフォーマンス最適化**: 入力変更時のプレビュー負荷を抑えるデバウンス処理済み。

## 始め方

1. **ディレクトリへ移動**
   ```bash
   cd sns/next-buzz-template
   ```

2. **依存関係のインストール**
   ```bash
   npm install
   ```

3. **開発サーバーの起動**
   ```bash
   npm run dev
   ```

## カスタマイズのポイント

### 1. OGPのデザイン調整
`src/app/api/og/route.tsx` を編集して、生成される画像のデザインを変更できます。

### 2. メインロジックの実装
`src/app/page.tsx` の `generateResult` 関数内に、独自の診断ロジックや生成ロジックを記述してください。

### 3. テーマカラーの変更
`src/app/globals.css` の `--color-brand` 等を調整することで、アプリ全体の雰囲気を一瞬で変えられます。

---
> [!IMPORTANT]
> このテンプレートは「バズる」ことを目的としているため、UIのインパクト（アニメーション、グラデーション、フォントサイズ）を敢えて強く設定しています。
